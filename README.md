# tennis-academy-management-system
simple Tennis Academy management web application- gsoc prorposal
I propose to build a simple web application that helps small tennis academies manage their daily operations. Administrators and coaches will be able to register players, store player information, schedule training sessions, and record match results. Many small academies still use manual notebooks or spreadsheets, which leads to disorganization and lost data.  

The frontend will be built with HTML and CSS. The backend will use Python Flask and MySQL (which I will learn during the project). The complete project will be open-source and hosted on GitHub.
